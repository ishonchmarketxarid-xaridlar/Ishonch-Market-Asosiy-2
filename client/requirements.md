## Packages
zustand | State management for the shopping cart with localStorage persistence

## Notes
- Images are dynamic based on imageUrl provided in the product object.
- The UI is mobile-first with a fixed bottom navigation bar.
- Translations are handled entirely on the frontend using React Context.
- For Telegram checkout, we assume opening a direct link is the preferred behavior for the "Send Telegram message" requirement since we don't have a backend Telegram bot API specified in the routes manifest.
